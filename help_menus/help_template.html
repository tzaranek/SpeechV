Windows

These commands can be used in any application in Navigate mode.

Command Description
Focus <application> Switch focus to the application
Note: The application must be open
Launch <application> Launch the application
Note: "Firefox" and "Word" are officially supported
Terminate Close the current application without saving or confirming
Move Move the SpeechV window to the opposite corner
Resize left/right Move the current application into the left/right half of the screen
Resize up/down Move the current application into the top/bottom half of the screen
Resize full Make the current application fullscreen
