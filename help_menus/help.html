Windows

These commands can be used in any application in Navigate mode.

Command Description
Focus <application> Switch focus to the application
Note: The application must be open
Launch <application> Launch the application
Note: "Firefox" and "Word" are officially supported
Terminate Close the current application without saving or confirming
Copy Copy highlighted data
Paste Paste data in clipboard
Move Move the SpeechV window to the opposite corner
Resize left/right Move the current application into the left/right half of the screen
Resize up/down Move the current application into the top/bottom half of the screen
Resize full Make the current application fullscreen

Firefox

These commands can be used in Firefox when you are in Navigate mode.

Command Description
Follow Label links and input fields, then open matching link in current tab
Open Label links and input fields, then open matching link in new tab
Up/Down Scroll vertically
Page up/down Scroll vertically by a screen
Left/right Scroll horizontally
Top/bottom Scroll to top/bottom of the page
Zero/dollar Scroll to leftmost/rightmost
Zoom in/out Zoom in/out
Zoom default Set default zoom
Delete Close tab
Undo Reopen recently closed tab
Previous/next Select left/right tab
Refresh Reload current tab
Find Search for text on page
New tab Open a new tab
New window Open a new window
