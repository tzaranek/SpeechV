SpeechV Help

This is a complete list of commands in SpeechV. You can say "up" or "down" to scroll, or use the links below. To use the links, say "follow" to assign a label to each link, then say the corresponding label. Say "top" to return to the top of the page.

Web Browsing

Task: Google a term and read up on it

1.
focus firefox
Places firefox in front of all other applications. SpeechV will only accept we browsing commands if firefox is in front, so that command is usage is unambiguous


2.
search patent
Googles 'patent' from the current tab. 'patent' could be replaced with anything


3.
follow 'a' 'e'
This command pops up a unique letter combination in front of all clickable links. Dictating those letters out loud will click on the associated link.


4.
down
Scrolls down. Similarly, 'up' scrolls up


5.
find modern
Finds and highlights 'modern' on the current web page. Use 'find next' and 'find previous' to explore other occurrences


Task: Create and use a bookmark

1.
focus firefox
Don't forget to place firefox in the foreground


2.
search patent
Find a page to bookmark


3.
save bookmark


4.
show bookmarks
Display ALL bookmarks


5.
follow 'a'
Choose a bookmark to open just as you would with a google search using 'follow'


Task: Compare two web pages

1.
focus firefox


2.
search larceny


3.
open 'a' 'e'
This is like the 'follow' command where unique letters pop up next to each link. Here, though, the link is opened in a new tab


4.
search burglary


5.
tab left
Switch one tab to the left. 'tab right' works similarly


6.
back
Go back in page history. Similarly, 'forward' would return to the wikipedia page if used after this command

Full list of commands

Windows

These commands can be used in any application in Navigate mode.

Command Description
Focus <application> Switch focus to the application
Note: The application must be open
Launch <application> Launch the application
Note: "Firefox" and "Word" are officially supported
Terminate Close the current application without saving or confirming
Move Move the SpeechV window to the opposite corner
Resize left/right Move the current application into the left/right half of the screen
Resize up/down Move the current application into the top/bottom half of the screen
Resize full Make the current application fullscreen

Firefox

These commands can be used in Firefox when you are in Navigate mode.

Command Description
Follow Label links and input fields, then open matching link in current tab
Open Label links and input fields, then open matching link in new tab
Up/Down Scroll vertically
Page up/down Scroll vertically by a screen
Left/right Scroll horizontally
Top/bottom Scroll to top/bottom of the page
Leftmost/rightmost Scroll to leftmost/rightmost
Zoom in/out Zoom in/out
Zoom default Set default zoom
Close tab Close tab
Reopen tab Reopen recently closed tab
Tab left/right Select left/right tab
Back/forward Go back/forward in history
Refresh Reload current tab
Find Search for text on page
Find next Search for text on page
Find previous Search for text on page
New tab Open a new tab
Save bookmark Open a new window
Show bookmarks Open a new window
Search Open a new window
