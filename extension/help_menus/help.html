SpeechV Help

This is a the help guide for SpeechV. You can find recipes on how to perform common tasks, an explanation of the different modes in SpeechV and how to switch between them, instructions on troubleshooting voice recognition and a complete list of commands.
To navigate the page, you can say "up" or "down" to scroll, or use the links below. To use the links, say "follow" to assign a label to each link, then say the corresponding label. Say "top" to return to the top of the page.

Web Browsing Recipes

Task: Google a term and read up on it

1.
focus firefox
Places firefox in front of all other applications. SpeechV will only accept we browsing commands if firefox is in front, so that command usage is unambiguous


2.
search patent
Googles 'patent' from the current tab. 'patent' could be replaced with anything


3.
follow 'b'
This command pops up a unique letter combination in front of all clickable links. Dictating those letters out loud will click on the associated link. In this case, 'b' should be said to click on the wikipedia link, but the correct letter may vary search to search


4.
down
Scrolls down. Similarly, 'up' scrolls up


5.
find modern
Finds and highlights 'modern' on the current web page. Use 'find next' and 'find previous' to explore other occurrences


Task: Create and use a bookmark

1.
focus firefox
Don't forget to place firefox in the foreground


2.
search patent
Find a page to bookmark


3.
save bookmark


4.
show bookmarks
Display ALL bookmarks


5.
follow 'a'
Choose a bookmark to open just as you would with a google search using 'follow'


Task: Compare two web pages

1.
focus firefox


2.
search larceny


3.
open 'a' 'e'
This is like the 'follow' command where unique letters pop up next to each link. Here, though, the link is opened in a new tab


4.
search burglary


5.
tab left
Switch one tab to the left. 'tab right' works similarly


6.
back
Go back in page history. Similarly, 'forward' would return to the wikipedia page if used after this command


Word Processing Recipes

Task: Write in a Word document

1.
launch word
Launch word if it's not already open. Otherwise, use 'focus' instead


2.
follow 'L'
'follow' enumerates all clickable buttons with letters, just like 'follow' does with Firefox. In this case, 'L' will open a new document


3.
insert
Tell SpeechV that the following text should be written into the document and not interpreted as voice commands. SpeechV will continue to interpret all voice in this way until we say 'navigate' again


4.
Title
Say what you want to be the title


5.
Enter
If said alone, 'enter' will move the cursor onto the next line


6.
This is an example period
Saying 'period' at the end of a chunk of speech will end the text inserted with a '.'


7.
period
Like 'enter', saying 'period' alone will input a '.'


8.
navigate
Tell SpeechV that any speech should be interpreted as a SpeechV command, effectively taking us out of 'Insertion mode'


9.
bold
This will make it so any text entered will be bolded. Use 'bold' again to unbold any new text to be added. Note that 'bold' can only be used from 'Navigation mode'


10.
insert


11.
done
type 'done' into the doc to show how bold works


Task: Save a document with a new name

1.
launch word
Launch word if it's not already open. Otherwise, use 'focus' instead


2.
follow 'L'
'follow' enumerates all clickable buttons with letters, just like 'follow' does with Firefox. In this case, 'L' will open a new document


3.
insert
Tell SpeechV that the following text should be written into the document and not interpreted as voice commands. SpeechV will continue to interpret all voice in this way until we say 'navigate' again


4.
Example text
Insert some text into the doc so that we can demonstrate saving


5.
navigate
Tell SpeechV that any speech should be interpreted as a SpeechV command, effectively taking us out of 'Insertion mode'


6.
follow 'F' 'A' 'Y' '3'
Saving a document with a new name can be a bit clunky, but the general idea is to repeatedly follow clickable objects with 'follow'. The string of characters will place the cursor into a spot where the file may be named


7.
insert
Tell SpeechV that we want to enter text into word, so that we can name the file


8.
filename
Provide a filename for the document we're saving


9.
navigate


10.
follow 'A' 'Y' '5'
Use follow again to click on the save button and complete this task. Note that saving without specifying a new name is far simpler: "follow '1'"


Task: Edit a document

1.
launch word


2.
follow 'Y' '1'
Open the document you wish to edit. In this recipe, we open the most recently opened document


3.
insert; Title to edit; navigate
Enter some text to manipulate for this example. Semicolons separate commands here


4.
highlight
This tells SpeechV to interpret following 'left', 'right', 'up', and 'all' commands as verbs intended to highlight portions of the document.


5.
left; left; left
Highlight the title words. Left is said once for each word.


6.
bold
Bold the highlighted text. 'Italics', 'Underline', and 'Delete' are other useful verbs


7.
unhighlight
Deselects any highlighted text and places the cursor to the right of the previously highlighted text


App management Recipes

Task: Work with Word and Firefox side by side

1.
focus firefox
Places firefox in front of all other applications. SpeechV will only accept we browsing commands if firefox is in front, so that command usage is unambiguous


2.
resize left
Orient firefox such that it takes up the left half of the screen


3.
launch word
Opens Word if it's not already started. Otherwise, this will make Word the frontmost window


4.
resize right
Like 'resize left', except it Word will take the right half of the screen


5.
switch
Alt-tab once to switch to the most recently used application. There isn't a generalized alt-tab because we believe that 'focus' is far more attractive in the general case of switching between applications


Task: Hide applications

1.
focus firefox


2.
minimize
Hide but don't close the top application (in this case firefox)


3.
launch word


4.
terminate
Close Word WITHOUT saving changes. You MUST save changes before using this command. 'terminate' will not work for firefox because of (currently unsolvable) technical issues


Modes of operation in SpeechV

There are 4 modes of operation in SpeechV: navigation, insertion, standby and configuration.

Navigation

This is the main mode of operation in which SpeechV listens for and executes commands.

Insertion

In this mode, SpeechV performs dictation, and enters your speech as text in the application currently open. End phrases with 'period' or 'comma' to insert punctuation. Saying "enter" on its own will insert a new line, while saying "new paragraph" inserts two new lines.

To enter this mode from navigation, say "insert". To end insertion and return to navigation, say "navigate".

Standby

In this mode, SpeechV does not process your speech.

To enter this mode from navigation, say "sleep". To return to navigation, say "navigate".

Configuration

In this mode, SpeechV opens the settings panel and allows you to adjust your settings.

To enter this mode from navigation, say "settings". To close the panel and resume regular usage, say "close".

Troubleshooting voice recognition

If SpeechV is not working well, you are likely experiencing one of two problems:

1. SpeechV cuts you off while you're still speaking
2. SpeechV cannot recognize your voice or processes audio when you aren't speaking

Problem 1 can be fixed by accessing the settings panel. From there, increase the voice timeout setting. This is the amount of silence that signals the end of a command. Increasing audio timeout allows for longer pauses when speaking, but slows down overall performance.

Problem 2 is likely an issue with you environment. SpeechV tries to automatically detect and adjust for ambient noise, but consider using SpeechV in a quiet environment for optimal performance.

Windows

These commands can be used in any application in Navigate mode.

Command Description
Focus <application> Switch focus to the application
Note: The application must be open
Launch <application> Launch the application
Note: "Firefox" and "Word" are officially supported
Terminate Close the current application without saving or confirming
Move Move the SpeechV window to the opposite corner
Resize left/right Move the current application into the left/right half of the screen
Resize up/down Move the current application into the top/bottom half of the screen
Resize full Make the current application fullscreen
Settings Configure SpeechV
Record start Begin recording a macro
Record end Finish recording a macro
After ending a recording, a prompt to name the macro and a confirmation prompt will appear to complete the process
Copy Copy selected data
Paste Paste copied data
Switch Alt-tabs once, switching to the second most recently opened application
Minimize/maximize Minimizes/maximizes currently open application

Note: this command is only valid after executing "Find <phrase>"

Microsoft Word

These commands can be used in Microsoft Word when you are in Navigate mode.

Command Description
Up/down [X] Move the cursor up/down X lines
If X is not provided, default to 1
Left/right [X] Move the cursor right/left X words
If X is not provided, default to 1
Paragraph up/down Move the cursor to the beginning/end of paragraph, or to the paragraph above/below
Page up/down Move the cursor to the beginning/end of the page, or to the page above/below
Follow This is a multi-part command that allows you to access the toolbar:
1. "Follow" labels buttons in the toolbar
2. Say the label corresponding to the desired action
"Cancel" to exit the toolbar without selecting anything
"Back" to go back one layer
Highlight This is a multi-part command to highlight regions of text:
1. "Highlight" begins highlighting text.
2. "Up/down/left/right" [X] extends the highlighted region by X lines/words. If X is not provided, default is 1.
Alternatively, "all" selects all the text.
Unhighlight Unselect highlighted text
Cut Remove highlighted text from original position and copy to clipboard
Copy Duplicate highlighted text
Paste Paste cut/copied text
Undo/redo Undo/redo action
Bold/italics/underline Toggle on/off formatting
Left/center/right align Adjust alignment of text
Indent Insert indentation at cursor position
Remove indent Remove indentation at cursor position
Increase/decrease size Increase/decrease font size
Caps Cycles the highlighted text between UPPER CASE, lower case, and Title Case
Delete Delete character to the left of cursor
<punctuation> Insert punctuation at cursor position
Supported punctuation types: space, new line, period, comma, exclamation, question, slash, colon, semicolon, apostrophe, quote, open parenthesis, close parenthesis, ampersand, dollar, star
